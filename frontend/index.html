Ambiente Ativo
Guia de Uso
  1. Configure o ambiente: Copie .env.example para .env, gere uma SECRET_KEY Fernet e defina AMBIENTE=homologacao.
  2. Cadastre uma empresa: Clique no + ao lado do seletor de empresa no topo, informe o CNPJ, razão social, caminho do arquivo .pfx e a senha do certificado.
  3. Selecione a empresa ativa: Use o dropdown no cabeçalho. Todas as operações usarão o certificado desta empresa.
  4. Execute as operações: Navegue pelo menu lateral e preencha os formulários. Os XMLs são montados, assinados e enviados automaticamente.
  5. Analise os resultados: A resposta da prefeitura é exibida em XML formatado. Use os botões de copiar/baixar para salvar o comprovante.
